常見問題 FAQ

關於 OpenCRAW 的安裝、費用、安全與支援，我們整理了 33 項完整解答

【基礎認識篇】

Q1. OpenCRAW 是什麼？
OpenCRAW（前身為 Clawdbot/Moltbot）是一個開源的 AI 助理框架，讓你可以在自己的電腦或雲端伺服器上運行 AI 助理，整合 Telegram、Discord、WhatsApp 等平台，自動化處理各種工作流程。
Q2. 跟 ChatGPT 有什麼不同？
ChatGPT: 雲端服務，需要網路連線，資料儲存在 OpenAI 伺服器
OpenCRAW: 本地運行，資料存在你的電腦，隱私性更高，可客製化自動化流程
Q3. 我需要什麼技術基礎嗎？
基礎安裝不需要！我會幫你處理所有技術細節。但進階客製化需要基本電腦操作能力。

【安裝流程篇】

Q4. 安裝需要多久時間？
基礎安裝：約 30-60 分鐘；進階設定：約 1-2 小時；客製化開發：依需求 1-7 天不等。
Q5. 我需要準備什麼？
1. 電腦 (Mac/Win/Linux) 2. 穩定的網路 3. API Key (Gemini 免費版) 4. Telegram 帳號。
Q6. Windows 可以安裝嗎？
可以！但建議使用 WSL2 或 Docker 來獲得最佳體驗。Mac 的安裝會更順暢。
Q7. 安裝過程中我需要在場嗎？
建議在場！我會透過遠端協助工具（AnyDesk/TeamViewer）操作，你可以全程看到並學習。

【費用與付款篇】

Q8. 除了安裝費，還有什麼額外成本？
API 費用（依使用量）、雲端主機費（若需 24h 運行）、進階功能外掛費。
Q9. 可以分期付款嗎？
基礎方案需全額付款。進階以上方案可談分期（首付 50%，完成後 50%）。
Q10. 有提供發票嗎？
可以開立電子發票（個人或公司戶）。公司戶需另加 5% 稅金。

【技術支援篇】

Q11. 安裝後有保固嗎？
基礎方案：7 天；商務方案：30 天；豪華方案：一整年 VIP 維護。
Q12. 之後遇到問題怎麼辦？
保固期內免費處理。保固期外可單次付費支援（$500/次）或訂閱 VIP 維護方案。
Q13. 可以升級方案嗎？
可以！補差價即可升級。例如基礎方案升級進階方案。
Q14. 如果不滿意可以退費嗎？
安裝前全額退費；安裝中依比例退費；完成後因屬數位服務原則不退費，但會協助調整至滿意。

【功能與客製化篇】

Q15. 可以連接哪些平台？
支援 Telegram、Discord、WhatsApp、Slack、iMessage、Signal 等。
Q16. 可以同時使用多個 AI 模型嗎？
可以！我會幫你設定 /g3f, /k25, /thinking 等指令，讓系統依據任務性質自動切換。
Q17. 可以客製化自動化流程嗎？
可以！如自動整理截圖、自動下載 YT 生成簡報、監控電商訂單等。
Q18. 資料安全嗎？
資料儲存在本地，API Key 絕不外洩，可設定防火牆與權限控管。
